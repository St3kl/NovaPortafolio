# NovaPortfolio - Clean Portfolio Website Template

🚀 NovaPortfolio is a modern, clean, and fully responsive portfolio template designed for developers, designers, and digital creators who want to showcase their work with simplicity and style.

---

## 🌟 Features

- Fully responsive design (mobile, tablet, desktop)
- Clean and minimal UI
- Smooth scrolling navigation
- Project showcase section
- Skills display
- Contact form layout
- Easy to customize and beginner-friendly

---

## 🎯 Who It's For

NovaPortfolio is perfect for:
- Web developers
- UI/UX designers
- Freelancers
- Students building their first portfolio

---

## 🧩 Sections Included

- Hero (introduction + call-to-action)
- About (personal summary)
- Projects (work showcase)
- Skills (technologies/tools)
- Contact (simple form)
- Footer

---

## ⚙️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

---

## 📦 How to Use

1. Download or clone the template
2. Open `index.html` in your browser
3. Replace placeholder content with your own
4. Customize styles in `styles.css`
5. Deploy using GitHub Pages or any hosting platform

---

## 💡 Why NovaPortfolio?

This template focuses on simplicity and clarity, making it ideal for users who want a professional-looking portfolio without dealing with complex frameworks or setup.
